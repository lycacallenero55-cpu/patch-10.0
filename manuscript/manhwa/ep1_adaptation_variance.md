**RECORD — ACCEPTED (⑤ gate-verified CYCLE-0002; per QA-240)**

- **Author:** ② Lore Department, CYCLE-0001 (TASK-0008).
- **Scope:** compares `manuscript/chapters/00_prologue.md` + `manuscript/chapters/01_the_ninth_world.md` (MANUSCRIPT_CANON, prose) against `manuscript/manhwa/ep1_v3_script.md` (ADAPTATION_CANON — beats) and `readers/episode1_part1.html` + `readers/episode1_part2.html` (DERIVATIVE OUTPUT — current published wording), per `SOURCE_OF_TRUTH.md` classifications.
- **Nature of this document:** a factual comparison record, not a canon change. No prose, script, or reader file was edited to produce this table. Where variance is found, this record states which version is currently *published* (the readers), not which version *should* be canon going forward — that determination, if any is needed, belongs to a future creator decision, not to this record.
- **Wording precedence used:** per `SOURCE_OF_TRUTH.md` row "manuscript/manhwa/ep1_v3_script.md … dialogue superseded by readers' wording," wherever the script and the published readers disagree on exact dialogue, **the readers' wording is recorded as current**; the script's wording is recorded as the prior/superseded draft. Both are shown in the Notes column so the drift is visible, not hidden.
- **Handbook alignment:** this table's variance classes are the work-order's requested vocabulary (`SAME_EVENT / COMPRESSED / REORDERED / ADDED_IN_ADAPTATION / OMITTED_FROM_ADAPTATION / DIALOGUE_REWORDED`). Handbook §14 uses a parallel but not identical vocabulary (`DIRECT / COMPRESSED / EXPANDED / REORDERED_PRESENTATION / OMITTED / ADAPTATION_CONNECTIVE`) for the same purpose — the two are compatible (`SAME_EVENT` ≈ `DIRECT`; `ADDED_IN_ADAPTATION` ≈ `EXPANDED` or `ADAPTATION_CONNECTIVE`) and no beat below required inventing a class outside the requested six.
