## Licensing Note — why this library is Drive-resident (no image bytes in this repo)

The source folder contains **saved webpage captures of scanlation releases of the licensed commercial webtoon *Pick Me Up: Infinite Gacha*** (Redice Studio adaptation; original by Hermode) — i.e., third-party copies of copyrighted, commercially published artwork, obtained from a scanlation aggregator (the site's own watermarks and endcards are visible on the pages themselves).

This repository is **public**. Committing those page images here would publicly redistribute copyrighted material. Accordingly, by ④'s decision under this order:

1. **Zero image bytes are stored in this repository.** The library is fully **Drive-resident**: the images stay in the Owner's Drive folder, and this file is the studio's index to them.
2. **No raw Drive file ids or share links appear in this public file.** At index time the source folder's sharing was set to "anyone with link," so publishing ids here would be equivalent to publishing the content. The folder id and the complete per-file id manifest are on record in the private ops channel (①'s dispatch thread and ④'s OPS-0002 report).
3. Departments needing access: request the folder link from the Owner or ① Executive Producer. Files are named exactly as listed below, so any entry in this index can be located in the folder by name.

*Record-keeping recommendation (recommend-only, per registry practice): if the Owner wants in-repo copies, the prerequisites would be restricting the Drive folder's sharing and making this repository private, followed by a fresh Owner ruling; ④ did not take any of those actions.*

## C. Curation rationale

- **Found:** 357 items in the source folder; **85** of them are story-page strips across 7 captured chapters (51, 85, 86, 103, 142, 144, 189). **Imported into the repo: 0** (Licensing Note above — this is a licensing decision, not a technical failure; the folder was neither empty nor unreadable).
- **Reviewed:** 28 strips (4 per chapter, spread across each chapter's page range) — verified as real images and visually assessed segment by segment.
- **Shortlist optimization:** variety over volume — the 28 rows in §A deliberately cover action spectacle (85/86/144), quiet interiors and introspection (51-B, 189-B), environment-forward staging (103-B/C), tragedy/horror registers (103-A, 142-B/C/D), comedy timing (86-C, 189-A), and the series' signature diegetic card-UI grammar (86-D, 103-D, 142-D, 144-C/D) — the last being directly relevant to our registry-card device concept (see CANON_REGISTRY §2 "Registry cards").
- **Near-duplicates skipped:** consecutive pages within the same fight were sampled at intervals rather than exhaustively; site chrome excluded entirely.
- **For ③ Visual Production:** treat §A as the recommended first-open set. Everything remains subject to the REFERENCE-ONLY banner at the top of this file.
